⚠️ Migration Challenges Demonstrated

This project showcases various challenges you'll encounter when migrating to Spring Boot:

  • WAR packaging vs JAR packaging
  • JSP limitations in executable JARs
  • JNDI resource configuration
  • web.xml to Java config conversion
  • javax.* to jakarta.* namespace
  • Session-based to stateless architecture
  • Filter-based security to Spring Security
  • And more...
